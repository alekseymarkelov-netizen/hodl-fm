# HODL FM

Фоновое чувство крипто-рынка. Музыка меняется по тренду биткоина — без алёртов, ambient awareness.

**Live:** https://hodl-fm.siberia.fund

5 состояний по 1h % изменению BTC: FALLING / COOLING / STILL / RISING / SURGE.
Звук синтезируется локально через Web Audio API. Цена — CoinGecko API.
